ProxyWar / ProxyWar public release

This repository is the public technical snapshot for ProxyWar, an
experimental autonomous-agent layer built on the OpenFront game engine.

Highlights:

- External agents use the AgentObservation plus LegalAction protocol.
- Agent Card onboarding is available for user-owned agent services.
- The starter template is published at 0xNad/ProxyWar-starter-agent.
- Hosted beta and operator runbooks use placeholders only; real invite codes,
  endpoint tokens, local artifacts, and deployment secrets are intentionally not
  included in this repository.
